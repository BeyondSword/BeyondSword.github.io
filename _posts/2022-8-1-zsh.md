# 总结一下zsh环境的搭建
zsh相比很多linux原生的bash命令行，功能更为强大，配合 oh-my-zsh 这个插件简化了配置流程，有很多自定义的空间。以下展示基于 ubuntu20.04。

0. 确认是否已经安装了 zsh
```
cat /etc/shells
```
1. 安装zsh。
```
apt install zsh
```
2. 将zsh设为默认shell。
```
chsh -s /bin/zsh
```
